# Jouaux Traiteur

Site vitrine pour le service de traiteur "Jouaux Traiteur" réalisé avec React et Vite.

## Prérequis

- [Node.js](https://nodejs.org/) >= 18
- npm

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

## Lint

```bash
npm run lint
```

## Tests

```bash
npm test
```

## Build

```bash
npm run build
```
